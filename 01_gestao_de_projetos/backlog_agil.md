## 1. Definição

## 5. Definição de Pronto (DoD — Definition of Done)

Uma User Story é considerada **CONCLUÍDA** quando:

- [ ] O artefato foi criado/código foi escrito
- [ ] Foi revisado pelo próprio aluno (self-review)
- [ ] Foi salvo e commitado no repositório GitHub
- [ ] Está na pasta/formato correto conforme a estrutura do repositório
- [ ] Para código Python: executa sem erros e produz o resultado esperado
- [ ] Para documentação: está em Markdown, formatada e completa
